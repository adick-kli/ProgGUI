# ProgGUI - Professional AT32UC3A1512 Programmer

Eine moderne, professionelle GUI für das Programmieren von Atmel AT32UC3A1512 Mikrokontrollern unter Windows.

## 🎯 Features

- ✅ Einfache, intuitive Benutzeroberfläche
- ✅ Modulare, testbare Architektur
- ✅ Persistente Einstellungsverwaltung
- ✅ Detailliertes Logging mit Farbcodierung
- ✅ Automatische Validierung aller Eingaben
- ✅ Sichere Command-Builder (keine Injection-Risiken)
- ✅ Thread-basierte Programmierung (responsive UI)
- ✅ Konvertierbar zu Windows .exe (PyInstaller)

## 📋 Anforderungen

- **Python 3.9+** (für die Entwicklung)
- **Windows 10/11**
- **Atmel Studio 7.0** (für die Tools: atprogram, atbackend, avr32-objcopy)
- **Atmel-ICE Programmer** (Hardware)
